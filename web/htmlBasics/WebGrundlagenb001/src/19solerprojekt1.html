Projekt 1

Primzahl-Ermittlung

Mein erstes selbständiges Projekt, war ein Programm das, nach einer Eingabe des Benutzers, ermittelt ob die eingegebene Zahl eine Primzahl ist oder nicht.
Dieses Programm hab ich mit der Programmiersprache C geschrieben. Das Programm startet mit einer einfachen 'printf'-Funktion, mit der ich den Benutzer frage, welche Zahl er eingeben möchte. Dem folgt eine weitere einfache Funktion. Diesmal jedoch eine 'scanf'-Funktion. In dieser Funktion gibt das Programm dem Benutzer die Chance, eine beliebige Zahl einzutragen, die geprüft werden soll. Dannach überprüft eine for-Schleife, in der eine if-Bedingung steht, welche überprüft ob ein Rest übrig bleibt, wenn man die angegebenen Zahl durch jede Zahl von zwei bis hoch zur Hälfte der angegebenen Zahl teilt. Dem ensprechend wird mit einer weiteren printf-Funktion dem Benutzer geschrieben ob es eine Primtahl ist oder nicht.
