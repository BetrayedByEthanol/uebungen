Rust

The first project I decided to undertake was to learn how to program using the non-standard programming language named Rust. Rust is a multi-paradigm system language focused on safety, especially safe concurrency. Rust is syntactically similar to C++, but is designed to provide better memory safety while maintaining high performance.
Having only been programming for about 2 months, I decided to attempt to translate the programming exercises we've done in C to Rust. Using The Book, I worked my way through the first few chapters and example exercises to get a new user acquainted with the language. Feeling somewhat confident, I decided to attempt programming our Venetian Blind program in Rust.

Harmony

The second project I decided to undertake was a group project, encompassing two of the three apprentice groups. The premise of the project was to program a functioning app that would be used by our Company. The app would provide information pertaining to the company, individual apprentices, an internal chat function, and allow various organizational functions.

Feeling that this project was too broad and all-encompassing for programmers of our experience level, we decided to condense the app's functions to increase the chance of reaching a realistic end goal and product.
  • Basic Chat Function
  • Account Creation
  • Account Management
  • Moderation Functionality
  • Password
  • Group Chat
  • Add To Group Functionality
  • Remove From Group Functionality
  • Private Chat
  • Friends List
  • Ignore Function
  • Delivered-Sent-Received Icons
Using these functions we've started planning out the UI and Functions themselves, defining the return values and types, and we will continue working on this project every Thursday for about 3 hours until the end of January.
